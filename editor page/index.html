What is Showray?

Showray is a website that lets you build music-related release pages to share with your listeners. Those pages can include various streaming services (including but not limited to Spotify, Apple Music, TIDAL and YouTube Music), custom links such as official merch websites, and more.

We let users create a website and have full access to it, being able to change anything whatsoever, and host the websites anywhere.

Why not other websites?

Yeah, there are several alternatives, but one little catch. We're free. Open-source. No attribution. No limits. :)

Is it REALLY free?

Yes, really. You can create a music release page for absolutely free and host it anywhere. You do not need to pay us anything.

Want something more? If you need custom features, changes and/or something else, feel free to contact me for anything special. This may come at a price. Alternatively, support me (donate) with Ko-Fi and request a feature.

Where is the code?

In the GitHub repository, silly. The repository includes version patch notes, full code, and the ability for anyone to post GitHub Issues.
